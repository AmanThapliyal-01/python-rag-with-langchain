# python-rag-with-langchain
Built a rag system in python for learning purpose


To run these run commands in following sequence

Initalize - one time setup

uv init - used to initalize the project
uv venv - used to create a virtual env
then use .venv\scripts\activate - to activate the script
use uv add -r requirements.txt - to install all dependencies
after than run uv add ipykernel to install the runtime kernel for this project.
